# Prosfata Inc. Website

Public website for Prosfata Inc., a boutique consulting and community support organization grounded in the UN Sustainable Development Goals.

Live at [prosfata.com](https://prosfata.com), served by GitHub Pages.

## Structure

- `index.html`, `community.html`, `contact.html`: the three pages
- `css/style.css`: single shared stylesheet (brand tokens, fluid type, layout)
- `js/main.js`: mobile navigation toggle only
- `assets/`: optimized images, icons, and self-hosted fonts
- `CNAME`: custom domain for GitHub Pages

Plain HTML, CSS, and vanilla JavaScript. No frameworks, no build step. Edit a file, commit, push, and the live site updates.
